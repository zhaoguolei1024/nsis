## 设置当前显示的TAB页

nsNiuniuSkin::ShowPageItem $hInstallDlg  "wizardTab" ${INSTALL_PAGE_CONFIG}

 

| 参数序号 | 参数类型 | 参数说明                     | 备注                                         |
| -------- | -------- | ---------------------------- | -------------------------------------------- |
| 1        | 整型     | 用于指定要设置标题的窗口句柄 | 调用init时返回  （可传递子窗口弹窗口的句柄） |
| 2        | 字符串   | 指定的TAB控件的name          |                                              |
| 3        | int      | 需要显示的tab页序号          | 以0为初始值                                  |
