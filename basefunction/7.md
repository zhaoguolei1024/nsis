## 显示界面控件

nsNiuniuSkin::ShowPage 0

当界面绑定事件完成后，通过此接口来调用显示控件，此接口将会阻塞运行。

| 参数序号 | 参数类型 | 参数说明                       | 备注                  |      |
| -------- | -------- | ------------------------------ | --------------------- | ---- |
| 1        | 整型     | 用于指定显示前要回调的NSIS函数 | 如果为0，则表示不回调 |      |
|          |          |                                |                       |      |
